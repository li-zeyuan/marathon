# 闭包

### 定义

- 官方：是由函数和其相关的引用环境组合而成的实体

### 闭包和变量逃逸

- ```go
  func incr() func() int {
  	var x int
  	return func() int {
  		x++
  		return x
  	}
  }
  
  func main() {
     i := incr()
  
     println(i()) // 1
     println(i()) // 2
     println(i()) // 3 
  }
  ```

- i就是一个闭包，i中有一个指针指向x

- x离开了作用域还存在，也是逃逸了

### 闭包引用

- ```go 
  x := 1
  func() {
  	println(&x) // 0xc0000de790
  }()
  println(&x) // 0xc0000de790
  ```

- 闭包内对外层变量是引用
